== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload all the dir wp-reciprocal-links/ to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a Page or a Post and place [LINKEXCHANGE] Where you want
4. Enjoy the fully working form :)
